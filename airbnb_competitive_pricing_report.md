## Practical automation approach (HTML/JS)

A direct browser script can read Airbnb calendar cells by `data-testid="calendar-day-MM/DD/YYYY"` and `data-is-day-blocked`:

```js
const days = [...document.querySelectorAll('[data-testid^="calendar-day-"]')].map(el => ({
  date: el.getAttribute('data-testid').replace('calendar-day-', ''),
  blocked: el.getAttribute('data-is-day-blocked') === 'true'
}));

const byDate = new Map();
for (const d of days) byDate.set(d.date, d.blocked);

const normalized = [...byDate.entries()].map(([date, blocked]) => ({ date, blocked }));
const booked = normalized.filter(d => d.blocked);
const available = normalized.filter(d => !d.blocked);
```
